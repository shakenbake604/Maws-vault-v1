A comprehensive Obsidian vault template designed for me but can be optimized for your workflow too.It features a beautiful interface with custom themes, powerful organization systems, and productivity-enhancing plugins.

## Features

- **Organization Systems**: Pre-configured folders for Articles, Projects, School, Job Applications, and more
- **UI**: Velocity theme with custom banners, covers, and styling
- **Dynamic Views**: Multiple bases views including charts, timelines, calendars, and feeds
- **Task Management**: task tracking with progress visualization and heatmaps using dataview and tasknotes
- **Content Management**: Bases CMS for managing notes like a content management system
- **Quick Note Creation**: Templates for projects, ideas, classes, and daily notes
- **Powerful Search**: Omnisearch plugin for finding anything instantly
- **Flashcards**: flashcard plug-in with learning modes

## Getting Started

1. **Download the Vault**
   ```bash
   git clone https://github.com/Maws7140/Maws-vault-v1
   cd "Maws vault v1"
   ```

2. **Open in Obsidian**
   - Launch Obsidian
   - Click "Open folder as vault"
   - Select the "Maws vault v1" folder

3. **Install Community Plugins**
   - When prompted, click "Trust author and enable plugins"
   - Or go to Settings → Community Plugins → Turn on community plugins
   - The vault will automatically install all required plugins

4. **Set Homepage** (Optional)
   - Go to Settings → Community Plugins → Homepage
   - Set homepage to: `Meta/Dashboards/Home..md`

## Themes

This vault uses the **Velocity Theme**. The theme is included and will be automatically applied when you open the vault.

## Configuration

### Customizing Templates

Templates are located in `Meta/Templates/`. You can customize:
- `newprojecttemp.md` - New project template
- `ideatemp.md` - Idea note template
- `newnotetemp.md` - General note template

### Modifying Dashboards

Main dashboards are in `Meta/Dashboards/`:
- `Home..md` - Main homepage with search and navigation
- `Notion style home.md` - Alternative Notion-style homepage

### Adjusting Bases Views

Base configurations are in `Meta/Bases/` and can be customized to display your data differently.

## Usage Tips

1. **Daily Notes**: Use Ctrl/Cmd+P → "Open today's daily note" to start your day
2. **Quick Capture**: Use the buttons on the homepage to quickly create new notes
3. **Search**: Press Ctrl/Cmd+Shift+O for Omnisearch
4. **Tasks**: Create tasks with `- [ ] Task name 2024-12-31` syntax
5. **Templates**: Use Ctrl/Cmd+P → "Templater: Insert template" anywhere
